# Practice Backlog

Use this folder to implement each exercise from `Java21_Concurrency_Complete_Guide.md`.

## Suggested file naming

- `Exercise01ThreadBasics.java`
- `Exercise02RaceConditionSync.java`
- `Exercise02RaceConditionAtomic.java`
- `Exercise02RaceConditionLock.java`
- ...

## Suggested order

1. Section 19 beginner exercises (1-3)
2. Section 19 intermediate exercises (4-7)
3. Section 19 advanced exercises (8-15)
4. Section 20 mini-projects

## Tip

Keep one class per exercise and a small `main` method so you can run each quickly while learning.
